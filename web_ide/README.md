# simflow web ide

## Development

Install dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build locally:

```sh
npm run preview
```

## Project structure

- `src/`: application source code
- `index.html`: Vite entry HTML
- `vite.config.ts`: Vite configuration
